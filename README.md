# 2026 Emerging Journals Database

A fully searchable, self-contained web page listing 22,281 academic journals 
from the 2026 Emerging Journal Quartile Rankings (新锐分区), translated into English.

## Features
- Search by journal name or ISSN
- Filter by quartile: Q1, Q2, Q3, Q4
- Filter by category (21 disciplines)
- Sort by rank, name, quartile, or category
- No backend required — runs entirely in the browser

## Categories Covered
Agriculture & Forestry · Arts · Biology · Chemistry · Computer Science · 
Earth Sciences · Economics · Education · Engineering & Technology · 
Environmental Science & Ecology · History · Literature · Management Science · 
Materials Science · Mathematics · Medicine · Multidisciplinary · Philosophy · 
Physics & Astrophysics · Psychology · Sociology

## Data Source
Data compiled by the **Bajwa**.  
This project presents the data in an accessible English-language interface.
